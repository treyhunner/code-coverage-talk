What is code coverage?

“In computer science, code coverage is a measure used to describe the degree to which the source code of a program is tested by a particular test suite.” — Wikipedia

Code coverage: in practice

Run my code and measure which parts were utilized.

From the perspective of a code coverage tool, code coverage means: run my code and record which parts of the code were used.

Popular Coverage Metrics

  • Statement coverage or line coverage
  • Function coverage

What is it good for?

  • 100% coverage is not an indicator of good tests
  • < 100% coverage tells us we missed something
Code coverage is a metric. Metrics are useless unless we understand their purpose. In the game of code coverage, 100% is the magic number. If our tests execute every edge of our code, they will have 100% coverage. However, 100% code coverage does not mean our tests are comprehensive or well-written. Code coverage says nothing about our test assertions and it does not measure the interdependence of code branches.

JS Code Coverage Tools

  • Istanbul: requires Node.js
  • Blanket.js: can run directly from web browser
  • JSCover: requires Java
Istanbul needs to modify your code before running it so you'll need Node.js installed to run your tests. Blanket.js can be run through Node or included directly in your web browser. JSCover requires Java and I have never used it.

Gamifying Coverage

  • Continuous Integration: did my tests break?
  • Coverage reporting: did my code coverage decrease?
Just as continuous integration services notify you when you break your tests, code coverage reporting services notify you when your code coverage decreases.

Code Coverage Reporting

  • Coveralls: popular and widely used
  • Codecov: supports branch coverage and partial line coverage
Coveralls is the current big fish in this space. As far as I know, they were the first SaaS code coverage product. Codecov is the new kid on the block. Coveralls does not support branch coverage or statement coverage (it sees everything in lines). Codecov supports both branch coverage and statement coverage. I found Codecov last year while looking for a solution to Coveralls' lacking support for branch coverage. Both Coveralls and Codecov are free for open source projects. Coveralls starts at $5 per repository and Codecov starts is $5 for all private repositories in your account.
